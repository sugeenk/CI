***************SELENIUM EXCEPTIONS
NoSuchElementException ( element locator may be incorrect)
WebdriverException - element is not clickable ( when we are not able to click on element, JS can resolve this issue)
StaleNotVisibleExcption  ( element is  not on DOM , it can be syncronization issue) 
ElementNotVisibleException 
ElementNotInteracctableException  (it can be syncronization issue )
NoSuchFrameException 
NoSuchWindowException
InvalidSelectorException  ( way to write locator may be incorrect eg. xpath = //@id='value')  )
InvalidElementStateException ( when element  are present on dom with disabled state)
